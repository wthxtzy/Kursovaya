# 🎮 Проект: Цифровой магазин игр «GAME STORE» (C# WinForms)

**GAME STORE** — это полноценное настольное приложение для управления игровым магазином, разработанное на языке C# с использованием технологии Windows Forms и интеграцией с базой данных. Проект разделен на две основные роли: Администратор и Пользователь, каждая из которых обладает уникальным функционалом и стилизованным темным интерфейсом.

## 🛠 Технологический стек
*   **Язык:** C# (.NET Framework)
*   **БД:** Microsoft SQL Server (DataSet, TableAdapters)
*   **Интерфейс:** Windows Forms (Custom UI, GDI+ для градиентов и кастомной отрисовки)
*   **Интеграции:** HttpClient (API qBittorrent WebUI), System.Text.Json

---

## 🚀 Основной функционал

### 👤 Режим пользователя
Интерфейс для покупателя, ориентированный на удобную навигацию и покупку контента.
*   **Магазин (Главная):** 
    *   Динамический список игр с поиском в реальном времени.
    *   Просмотр подробной информации: обложка, цена, дата выхода и средний рейтинг.
    *   **Всплывающие подсказки:** при наведении на жанр отображается его описание из базы данных.
*   **Система оценки:** Пользователь может выставлять оценку (1-5 звезд) любой игре. Средний рейтинг в магазине пересчитывается автоматически.
*   **Корзина:** 
    *   Добавление нескольких товаров.
    *   Автоматический расчет итоговой суммы и количества позиций.
    *   Возможность как одиночной покупки («Купить сейчас»), так и массового оформления заказа.
*   **Библиотека:** Список всех купленных игр с возможностью удаления.
*   **Профиль и статистика:** Раздел настроек отображает общую сумму потраченных средств, количество уникальных игр в коллекции и среднюю стоимость одной игры.
*   **Адаптивный дизайн:** Интерфейс полностью отзывчив и корректно перестраивается при переходе в полноэкранный режим.

### 🛡 Режим администратора
Панель управления для модерации контента и пользователей.
*   **Управление играми:** 
    *   Полное редактирование данных в таблице (Название, Жанр, Цена, Год).
    *   **Менеджер изображений:** Загрузка обложек для игр напрямую из файловой системы и их сохранение в БД в бинарном формате.
*   **Управление пользователями:** Просмотр и редактирование учетных записей.
*   **Контроль библиотеки и рейтингов:** Прямой доступ к управлению связями «пользователь-игра» и модерация оценок.
*   **Синхронизация:** Глобальная кнопка сохранения изменений, отправляющая все правки из DataSet обратно в SQL-базу.

## ⚡ Секретные функции (Easter Eggs)

В приложении реализована скрытая панель управления загрузками для продвинутых администраторов.
*   **Вызов:** Комбинация клавиш `Ctrl + Shift + 8`.
*   **Функционал:** Удаленное управление клиентом **qBittorrent** через WebUI API.
    *   Авторизация в торрент-клиенте.
    *   Загрузка файлов `.torrent` через интерфейс магазина.
    *   Мониторинг прогресса в реальном времени с кастомной отрисовкой Progress Bar в таблице.
    *   Управление скоростью и состоянием (Пауза/Запуск/Удаление).

## 🎨 Особенности интерфейса
*   **Темная тема:** Использование цветовой палитры Material Design (Dark Grey, Accent Green, Gold).
*   **Защита данных:** Пользовательский интерфейс имеет строгие ограничения (ReadOnly таблицы, запрет на изменение размеров колонок мышью), что предотвращает случайную порчу структуры таблиц.
*   **Безопасность:** Разграничение прав доступа при входе в систему.
*   **UX:** Использование ComboBox в таблицах для отображения текстовых названий жанров вместо их числовых ID.

---

## 📂 Структура базы данных
1.  **Games:** Хранит информацию об играх и обложки (Image).
2.  **Genres:** Справочник жанров с описаниями.
3.  **Users:** Данные учетных записей и баланс.
4.  **Cart:** Временное хранилище выбранных товаров.
5.  **Library:** Таблица владения (связь многие-ко-многим между Users и Games).
6.  **Ratings:** Оценки пользователей для формирования рейтинга.
